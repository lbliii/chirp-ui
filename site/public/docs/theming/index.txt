# Theming

URL: /docs/theming/
Section: theming

--------------------------------------------------------------------------------

Theming chirp-ui uses prefers-color-scheme for dark mode. Override any --chirpui-* variable to customize. Base Variables :root { --chirpui-accent: #7c3aed; --chirpui-container-max: 80rem; } Base colors drive derived states (hover, active, light, muted) via color-mix(). Manual Light/Dark Toggle Set data-theme=&quot;light&quot; or data-theme=&quot;dark&quot; on &lt;html&gt;. Optional Themes &lt;link rel=&quot;stylesheet&quot; href=&quot;/static/themes/holy-light.css&quot;&gt; Motion All animations respect prefers-reduced-motion. No configuration needed.
