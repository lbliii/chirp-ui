# Effects

Decorative backgrounds and ambient motion. Templates include `aurora.html`, `grain.html`, `constellation.html`, `particle_bg.html`, `meteor.html`, `orbit.html`, `dock.html`, `sparkle.html`, `confetti.html`, `wobble.html`, `symbol_rain.html`, `holy_light.html`, `rune_field.html`, `scanline.html`, and related hero wrappers.

## Variants

Many effects register under `VARIANT_REGISTRY` (e.g. `aurora`, `grain`, `orbit`, `sparkle`, `confetti`, `meteor`, `particle-bg`, `dock`). Sizes apply where listed in `SIZE_REGISTRY` (`orbit`, `sparkle`, `dock`, …).

## Performance

Prefer CSS-driven effects; respect `prefers-reduced-motion` via tokens — see [Motion](../theming/motion-and-transitions.md).
