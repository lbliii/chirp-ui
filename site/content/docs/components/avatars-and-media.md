# Avatars and media

| Template | Role |
|----------|------|
| `avatar.html`, `avatar_stack.html` | User avatars and stacks |
| `media_object.html` | Classic media + body row |
| `video_card.html`, `video_thumbnail.html` | Video previews |
| `carousel.html` | Sliding media |
| `post_card.html` | Feed-style cards |
