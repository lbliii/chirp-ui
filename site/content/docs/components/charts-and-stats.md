# Charts and stats

| Template | Role |
|----------|------|
| `bar_chart.html` | Bar visualization |
| `donut.html` | Donut chart |
| `stat.html` | Stat block |
| `metric_grid.html` | Grid of `metric_card` tiles |
| `animated_stat_card.html` | Animated emphasis |
| `animated_counter.html` | Number tickers (`number-ticker`, `animated-counter` variants) |
| `bento_grid.html` | Equal columns + default surface chrome (distinct from `grid()` presets) |

## Tokens

Charts respect `--chirpui-*` colors; use [Color system](../theming/color-system.md) for semantic colors.
