# Social and chat

| Template | Role |
|----------|------|
| `message_bubble.html` | Chat bubbles (`message_bubble` variants: default, user, assistant, system) |
| `message_thread.html` | Thread container |
| `conversation_list.html`, `conversation_item.html` | Inbox-style lists |
| `chat_input.html` | Composer |
| `typing_indicator.html` | “…” typing state |
| `comment.html` | Threaded comments |
| `reaction_pill.html` | Quick reactions |

## Layout

Use `chat_layout.html` with `fill=true` for full-height chat — see [Vertical layout](../guides/vertical-layout.md).
