## Get Started with chirp-ui

chirp-ui is an optional companion for Chirp. Install it when you want the component library, default design language, and app-shell patterns.

Two ways to adopt it:

1. **In a Chirp app** — install the library and call `use_chirp_ui(app)` to get the Kida macros, filters, and assets. Start at [[/docs/get-started/installation/|Installation]].
2. **On a Bengal site** — apply the packaged **chirp-theme** by setting `theme: "chirp-theme"` in your site config. Requires Bengal **>=0.3.3** (for `library_asset_tags()`). See [[/docs/get-started/installation/#apply-the-chirp-theme-bengal-theme|Apply the chirp-theme Bengal theme]] and the [[/docs/theming/chirp-theme/|chirp-theme reference]].
