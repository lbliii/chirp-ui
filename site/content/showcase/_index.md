# Component showcase

The full interactive gallery of every chirp-ui macro and pattern is generated by
`uv run poe docs-build-all` (or `make showcase-public`) and served at this route —
it replaces this page's HTML output with the assembled static showcase.

If you are reading this copy on the published site, the gallery injection step did
not run. The live specimens below are a small, self-contained preview so this page
still demonstrates real components even when the gallery is not injected. To build
the full gallery offline, run `make showcase` and open `_site/index.html`.
