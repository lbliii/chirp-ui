## Added

The docs site now has **learning tracks** — ordered reading paths that stitch
existing docs pages into a guided sequence with in-track progress and prev/next
navigation. The first track, *Getting Started*, walks a new reader from install
through theming and the layout vocabulary.

## Notes

Tracks do not duplicate content; following one is reading the same docs pages in
a curated order. See the [Tracks](/tracks/) section.
